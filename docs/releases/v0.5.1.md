新增
--

*   **终端扩容模组**：机器人控制终端专用调校升级，安装后可部署机器人上限从 3 提升至 5
    *   通过精密调校仪右键控制终端打开升级页面，放入终端扩容模组即可生效
    *   移除模组后超出新上限的机器人自动召回并弹出物品
*   **终端 GUI 改版**：新增顶部「终端概览」状态栏，一目了然显示区域大小、出货/充汽点状态、已部署机器人数
    *   机器人槽位根据当前上限动态显示，超出上限的槽位以红色玻璃板锁定并提示升级
    *   出货/充汽点绑定按钮现显示三种状态：有效（绿色坐标）、已失效（红色提示 + 删除线坐标）、未绑定

变更
--

*   **出货/充汽点自动失效检测**：当绑定的出货容器或充汽舱方块被破坏时，终端在下次访问时自动标记为失效，无需手动解绑
*   **机器人默认上限调整**：控制终端基础可部署机器人数从 4 调整为 3，通过终端扩容模组可升至 5
*   **控制终端支持精密调校**：终端现实现 UpgradeableMachine 接口，可通过调校仪右键打开升级槽

修复
--

*   **拾取机器人黑名单不过期**：拾取接近上限过短（40 tick）导致掉落物被反复加入/移出黑名单，机器人永远不去捡——改为独立 200 tick 接近上限，并在重置工作时清除黑名单
*   **机器人 MSPT 性能优化**：
    *   名牌更新改为脏标记模式，仅在活动状态变更时写入，避免每 tick 构建 Component
    *   拾取扫描增加快速路径：正在追踪的目标仍有效时跳过全量实体扫描（O(1) 查找）
    *   屠宰扫描增加空闲冷却门控，无目标时不执行昂贵的 `getNearbyEntities`
    *   农耕/采矿路线排序中 `Location.distanceSquared` 替换为内联 `dx²+dy²+dz²`，消除临时 Location 对象分配
