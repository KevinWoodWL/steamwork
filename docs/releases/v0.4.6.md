新增
--

*   **汽动逻辑 × 产线桥接**：新增 2 台桥接机器，让汽动逻辑系统与蓝图产线<b>双向</b>打通——产线事件可驱动逻辑、逻辑可反控产线物品流。两者需<b>同时</b>解锁「蒸汽货运」与「汽动逻辑」两项研究方可使用
    *   **气动产线传感器**（PneumaticLineSensor）：产线 → 逻辑。物品透传经过本机时按模式输出蒸汽信号——脉冲（每件触发）/ 堵塞（产线 jammed）/ 空闲（超时无物品经过）；信号灌入自身缓存，供相邻汽动逻辑元件读取
    *   **气动产线阀**（PneumaticLineValve）：逻辑 → 产线。信号面蒸汽 ≥ 阈值时放行缓存物品到下游，低于阈值时暂存；阈值、信号面、蒸汽种类均可在 GUI 调节

变更
--

*   **Waila 全面统一为 Pylon 风格**：所有机器 `getWaila()` 迁移到 `WailaDisplay.of(this, player).add(...)` 链式构造，流体/蒸汽进度条统一改用 `ProgressBar`，移除自实现的 `SteamworkUtils.createFluidAmountBar`
*   **流体 Waila 末尾显示流体名**：蒸汽/液体进度条改用 `ProgressBar.fluidContentsWithName`，末尾带 `(普通蒸汽)`/`(水)` 等括号标注，与 Pylon 原版表现一致
*   **研究树补全**：汽动脉冲器、汽动差压门、汽动锁存器加入「精密汽动逻辑」研究解锁列表；气动产线阀/传感器归入「蒸汽货运」研究，并登记物品、合成配方与货运指南页
*   **桥接机器双研究硬门控**：气动产线阀/传感器的放置需同时具备蒸汽货运（领取/合成）与汽动逻辑（放置）两项研究；其中汽动逻辑为硬门控，不受 `rebar.item.*` 绕过权限影响（OP / 创造亦受限），并在缺研究时给出提示
*   **产线兼容磨石手动操作**：`PylonLineOutputBridge` 新增后备路径——玩家手动操作处于产线中的磨石时，其产物（仅机器自然掉落）也能被识别并送入产线
*   **产线缓冲箱 GUI 提示**：缓冲箱界面新增槽位与模式说明项

修复
--

*   **汽动管道模块显示问题**（三处）：① 显示实体落在不透明木桶内部、环境光为 0 导致管道<b>发黑</b>——改为取相邻方块环境光设置亮度；② 连接输入 / 输出端时导管<b>伸出过长穿进端点连接桩</b>——前端收回到连接桩位置；③ 未连接时导管略微探出木桶面，避免与木桶材质冲突
*   **汽动导管口型 / 拐角连接穿模**：拐角连接时导管段前端越过方块边界探进相邻方块、造成材质重叠——前端收回到方块边界
*   **蒸汽冲压机活塞复位错位**：下压复位后活塞停在 Y=0.3、而非放置时的 Y=0.2375，导致与材质冲突——抽出单一常量统一放置 / 下压 / 复位三处坐标
