新增
--

*   **阶段 3 汽动逻辑扩展**：新增 5 台机器，补全逻辑层最后短板——差压比较、脉冲整形、SR 锁存、可编程差分序列、以及导管内嵌流量监控
*   **汽动差压门**（PneumaticDifferentialGate）：读取源 A（正后）与参考 B（右侧）的蒸汽量差值，A − B ≥ 阈值时放行蒸汽到正前输出面；防止回流浪费，适合梯度供汽
*   **汽动脉冲器**（PneumaticPulser）：从正后持续充汽，蓄能达阈值后向正前释放定量脉冲蒸汽；脉冲量与阈值均可在 GUI 调节，顶部中继器显示蓄能进度
*   **汽动锁存器**（PneumaticLatch）：SR 锁存器——左侧 SET 有汽锁住、右侧 RESET 有汽解锁；锁住时正后供汽持续流向正前输出；无信号时维持上次状态，RESET 优先
*   **蒸汽差分机**（SteamDifferenceEngine）：致敬巴贝奇差分机的可编程最多 3 阶差分寄存器；每个工作冲程消耗蒸汽，把各阶差分逐级累加生成序列；输出三选一：0–15 模拟红石 / 蒸汽脉冲量 / 分频脉冲间隔
*   **汽动管道模块**（PneumaticPressureModule）：嵌入汽动导管链中段，三种模式：流量计（过站物品速率→0–15 红石）、限流阀（限制每窗口通过物品数）、溢流报警（下游满载时输出红石）

变更
--

*   **汽动逻辑门重构**：逻辑门改为有朝向的比较器式布局——正前输出、正后/左/右三输入（未连接自动忽略），自带蒸汽缓存可门→门串联；新增蒸汽种类过滤（全部 / 普通 / 过热 / 加压）；Material 由铁块改为打蜡切制铜台阶
*   **蒸汽振荡器输入改为面直连**：振荡器不再走导管网络的 FluidPoint 输入端，改为 tick 主动从可配置的输入面抽取蒸汽；同时兼容 FluidTankRebarBlock（储罐）；新增 charge-rate 参数
*   **截止阀视觉升级**：截止阀核心由红石块 ItemDisplay 改为红石灯 BlockDisplay，开时熄灭、关时点亮，一眼可辨工作状态
*   **汽动输出端多端轮询抽取**：多个汽动输出端接入同一来源容器时，按服务器 tick 严格轮流抽取，避免先 tick 的把源抽空
*   **物品传输路径感知**：汽动输出端、分拣器、分配器在每次物品推送后通知路径上经过的管道模块，支撑流量计/限流阀/溢流报警功能
*   **蒸汽装备全面 i18n 化**：蒸汽护甲飞行模式（喷气推进 / 悬浮 / 关闭）、套装技能冷却、护盾吸收、蒸汽罐标签等硬编码中文全部改为 translatable 键，支持英文客户端
*   **钨套装地震伤害微调**：近处最高伤害 15→18、边缘最低 5→6，提升感知
*   **指南页面调整**：装配台移至装备子页面；蒸汽臂移至自动化页末尾；差分机、管道模块加入对应子页面
*   **研究树更新**：差分机加入「精密汽动逻辑」研究；管道模块加入「精密气动物流」研究；新增「精密蒸汽装配」研究条目
*   **用语统一**：lore 文案中「气动」统一为「汽动」

修复
--

*   **蒸汽飞行模式切换键失效**：`steamwork.message.steam_flight.mode.off` 等飞行相关翻译键之前缺失，导致模式切换显示为空白；现已补全全部飞行消息翻译键
*   **蒸汽充汽室物品计数显示**：物品计数行由硬编码「件」改为 translatable，英文客户端不再显示中文
