新增
--

*   **蒸汽罐插槽系统**：武器 / 工具 / 盔甲不再内置蒸汽容量，改为通过「蒸汽装备改装台」安装可更换的蒸汽罐（黄铜 / 因瓦 / 钨三级容量），由充汽舱充能
    *   未装罐：降级半成品，属性显著降低、无蒸汽能力
    *   装罐 · 有汽：满血，基础属性 + 蒸汽增益
    *   装罐 · 耗尽：明显降级，失去全部蒸汽增益
*   **钨套三模式飞行系统**：钨蒸汽套装新增完整飞行系统，Ctrl+Shift（疾跑+潜行）循环切换三种模式，模式选择写入 NBT 持久化
    *   喷气推进：长按空格点火升空 2 秒后快速推进，落地前空中复推无需重新点火
    *   悬浮自由飞行：飞行中切入即接管为类创造自由飞行（低速稳定悬停、禁止冲刺加速），落地自动切回喷气推进
    *   关闭：空格恢复正常跳跃，不触发喷气
*   **下坠动量抵消**：高速下坠时复推先用恒定反推力顶住下坠（带起飞式粒子特效），抵消时长随坠速增长，再转入推进
*   **钨套下坠缓降**：未在飞行且空中明显下坠时自动获得缓降保命，平常不给
*   **蒸汽装配祭坛**：竖直 3×3 多方块（中心装配台 + 8 个锰钢块底座），由背后管道供过热蒸汽驱动
    *   右键底座放料（手持右键放 1、潜行+右键全放、空手右键取回），物品像展示框那样悬浮显示
    *   右键装配台打开 GUI：底座镜像、过热蒸汽量表、状态、成品预览与合成按钮
    *   多条配方匹配同一组原料时点击成品预览循环切换；底座多放无关物品也能识别，且不消耗
*   **蒸汽飞行核心**：经催化反应、熔炼、精密铣削与装配四道工序锻造的飞行动力核心，是钨甲飞行的关键门槛
*   **新机械构件**：喷射喷嘴、涡轮转子
*   **蒸汽斧伐木**：斧子改为沿连通原木整棵砍（涵盖全部木质原木类型），不再误挖其它方块
*   **蒸汽工具被动增益**：手持有汽工具时——镐 / 铲给予急迫、斧给予力量，等级与时长随材质提升

变更
--

*   **装备能源统一为蒸汽罐**：所有蒸汽装备移除内置蒸汽容量设置，统一改为安装蒸汽罐供能；未装罐时为降级半成品
*   **高能机器需要加压蒸汽**：柴油等高能驱动机器改为需要加压蒸汽才能稳定输出
*   **钨套飞行物理重写**：飞行改由每 tick 引擎平滑驱动（取代每 0.5s 一次的脉冲），起飞、推进、悬浮的手感与耗汽全面调整
*   **整套钨盔甲配方门槛**：胸甲需要「蒸汽飞行核心」，头盔 / 护腿 / 靴子各需「喷射喷嘴」；武器工具保持简单不变
*   **中后期关键配方深化**：精密蒸汽涡轮改用「涡轮转子」、钨蒸汽罐追加钯合金锭 + 高压法兰、蒸汽充汽舱追加高压法兰
*   **范围采掘按工具区分**：镐只破坏需要正确工具的方块（不再挖泥土 / 沙子），铲只破坏与中心同种的方块（挖土只挖土、挖沙只挖沙）
*   **剑蒸汽喷射**：有效距离 4.5 → 8.0 格并与喷射粒子对齐；普通攻击与喷射的耗汽下调、命中粒子减量、声音调小
*   **钨套蒸汽地震**：近处伤害提高（最高 15，边缘最低 5）
*   **蒸汽剑基础攻击**：基础攻击伤害 7 → 6
*   **盔甲待机特效移除**：穿戴蒸汽盔甲待机时不再持续喷粒子与活塞声；飞行喷气的旋风射击音效大幅调小
*   **新增材质包绘制贡献者**：Takuya_Dongji（蒸汽工坊材质包主要绘制者）

修复
--

*   **创造模式飞行**：修复钨套飞行逻辑残留导致创造 / 旁观模式无法正常飞行的问题
*   **剑贯穿瞬间耗光蒸汽**：修复横扫 / 蒸汽喷射的内部伤害递归触发攻击处理器、两只生物连成一线时瞬间扣光蒸汽的问题
*   **蒸汽装配台多项修复**：改用自定义装配配方匹配（原先错误地匹配原版工作台配方导致完全不可用）、修复合成时 FLASH 粒子报错与「卡在装配中」、底座与 GUI 顺序对齐、流体类型改为过热蒸汽后可正常进汽
