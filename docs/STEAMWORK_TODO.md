# Steamwork To Do List

## 推荐推进顺序

1. 先收尾基础材料与机械组件：检查材质基底、删掉废弃项、补齐来源和用途。
2. 再打磨三台基础机器：蒸汽灭菌箱、蒸汽浸煮桶、蒸汽洗选槽，确保它们各自有稳定材料线。
3. 接着做第一批新蒸汽机器：优先蒸汽冲压机、蒸汽卷管机、蒸汽研磨机。
4. 然后推进合金与锅炉升级：让高级合金、过热蒸汽和多级锅炉真正进入主线。
5. 最后进入 Pylon / 柴油联动和终局倒推：深层矿浆、冷凝液、钯金催化框架、汽凝晶核。

## 下一次建议开工项

1. 检查所有基础材料的“来源 + 用途”，把没有用途的材料接到现有机器或机器配方里。
2. 重做三台基础机器的配方定位：浸煮桶负责有机/木材，洗选槽负责矿物湿法，灭菌箱负责生物质/橡胶/无菌材料。
3. 设计第一批“蒸汽装置/工具”，让 Steamwork 不只是在做加工机器，而是有更多像机械臂一样会改变自动化玩法的东西。

## 4. 蒸汽装置与工具线

- [x] 蒸汽机械臂：多输入、多输出、分配模式和目标列表。
- [ ] 蒸汽转运管夹：只负责从一侧容器按规则抽取到另一侧，不做长距离管道，适合小型产线拼接。
- [ ] 蒸汽分拣阀：按物品类型、标签或白名单把物品分到不同容器。
- [ ] 蒸汽装载台：把附近掉落物或指定方块产物收集进容器，作为机械臂的低阶补充。
- [ ] 蒸汽投送器：消耗蒸汽把物品投射到指定方向或目标容器，可用于跨缝隙物流。
- [ ] 蒸汽起重钩：移动方块或容器位置，偏工程建设工具，不直接复制物品。
- [ ] 蒸汽采集臂：对作物、浆果、甘蔗、竹子等进行定向收割，避免和普通矿机重叠。
- [ ] 蒸汽播种器：从容器取种子并补种，和采集臂组成早期农业自动化。
- [ ] 蒸汽压路锤：玩家手持工具，用蒸汽耐久/能量快速压实泥土、沙砾或部分方块。
- [ ] 蒸汽扳手：旋转、配置、绑定 Steamwork 方块和装置，也可作为 GUI 快捷工具。
- [ ] 蒸汽信号继动器：用蒸汽压力状态输出红石信号，让锅炉/储汽/机器状态能接入红石自动化。
- [ ] 蒸汽压力表：可放置或手持查看附近蒸汽网络、锅炉状态和耗汽设备。

## 5. 新鲜机制候选

这些内容不按原版或 Pylon 的常规思路做，目标是让 Steamwork 有自己的记忆点。

- [ ] 压力脉冲网络：机器不只消耗固定蒸汽，而是可以接收“压力脉冲”。不同频率的脉冲触发不同模式，例如快速搬运、保守搬运、只搬运满组。
- [ ] 蒸汽记忆盘：记录一个机器或装置的配置，然后复制到同类装置。适合机械臂目标、分配模式、过滤器规则。
- [ ] 冷凝标记器：给物品打上临时“湿润/冷凝/灭菌/加压”状态，某些配方需要状态而不是只需要物品本身。
- [ ] 压力锁：一种不靠红石的安全门/容器锁，只有附近蒸汽压力达到条件才允许打开、取物或输出。
- [ ] 蒸汽共振器：周期性扫描附近机器的工作节奏，如果多台机器节奏同步，就降低耗汽或加速；节奏错乱则效率下降。
- [ ] 气压邮筒：把小批量物品装进压力胶囊，在两个已绑定的邮筒之间传送，距离越远耗汽越多。
- [ ] 冷凝镜：把附近热源、锅炉或涡轮的“废热”转成少量冷凝液，作为后续终局材料的一条副线。
- [ ] 蒸汽惯性轮：储存一段时间内的过剩蒸汽输出，短时间释放给机械臂、涡轮或冲压类装置，形成爆发式工作窗口。
- [ ] 雾幕发生器：消耗蒸汽生成局部雾幕，降低怪物索敌距离、保护农田或影响某些冷凝配方环境。
- [ ] 压力校准台：不是加工物品，而是给工具/组件调参数，例如机械臂每次搬运量、扳手模式、压力表扫描范围。
- [ ] 蒸汽声呐：消耗蒸汽探测地下矿浆、洞穴水体或热源点，给后续深层矿浆线提供探索玩法。
- [ ] 汽凝裂隙：终局后生成的短时空间裂隙，不直接刷矿，而是提供特殊环境，让冷凝液、矿浆或催化框架进行高阶反应。
