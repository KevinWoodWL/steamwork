# 蒸汽工坊（Steamwork）

**Pylon 附属 · Rebar 框架 · Paper/Purpur 1.21.x · 纯服务端**

> 一句话：Pylon 给了你齿轮和熔炉，蒸汽工坊在它上面造了一整座蒸汽朋克工业城。

---

## 它不只是"多几台机器"

Pylon 提供了一套坚实的基础：矿石翻倍、手工合金、基础加工。但当你想把 100 台机器串成全自动产线、用蒸汽压力控制阀门开关、让管道根据下游容量自动限流时——你需要的不是更多机器，而是一整套**工业基础设施**。

蒸汽工坊就是这套基础设施。它在 Pylon 之上构建了四个全新维度：

---

## 一、蒸汽动力：从燃烧到三相蒸汽

Pylon 的机器靠原版燃料或手摇驱动。蒸汽工坊引入了完整的**蒸汽能源链**：

```
燃料 → 锅炉 → 普通蒸汽 → 加热室 → 过热蒸汽 → 加压机 → 加压蒸汽
                                      ↘ 涡旋管 → 加压蒸汽 + 蒸馏水
```

三种蒸汽不通用、各有用途：

| 蒸汽类型 | 用途 | 获取 |
|---|---|---|
| **普通蒸汽** | 基础加工（压力炉、研磨机、压机） | 锅炉直出 |
| **过热蒸汽** | 高阶冶金（精密铸造、装配台、催化反应） | 加热室升温 |
| **加压蒸汽** | 汽动物流网络的驱动力 | 加压机 / 涡旋管分离 |

锅炉本身也分四阶：**青铜 → 因瓦 → 锰钢 → 钨**，产量递增、对应研究树逐步解锁。每阶锅炉的热效率、缓存容量、升温速度都在 YAML 里可调——服主改一个数字就能调平衡。

**涡轮系统**更进一步：7 种涡轮（基础加工涡轮、精密加工涡轮、液压涡轮、柴油涡轮、精密蒸汽涡轮、Pylon 万能涡轮、简易蒸汽涡轮）可以贴着机器放，消耗蒸汽为相邻机器加速——你不是在"发电"，你是在用蒸汽的动能直接推动齿轮转得更快。

---

## 二、汽动物流：蒸汽驱动的物品网络

Pylon 的物品搬运依赖原版漏斗或 LogisticSlot 手动连接。蒸汽工坊构建了一套完整的**加压蒸汽驱动的物品管道网络**——概念上借鉴 PneumaticCraft，但深度集成了蒸汽体系：

### 核心组件

| 组件 | 功能 |
|---|---|
| **汽动导管** | 物品传输骨架，自动检测相邻连接、动态渲染管段 |
| **汽动输入端** | 从相邻容器抽取物品进入管网 |
| **汽动输出端** | 从管网推送物品到目标容器；多端接入同一源时**自动轮询**防抢占 |
| **蒸汽分拣器** | 按物品类型/NBT 过滤，精确路由到指定方向 |
| **汽动分配器** | 六面轮询分发，均匀负载 |
| **汽动货运站** | 大型中转枢纽，多方块结构，多路吞吐 |
| **蒸汽弹射器** | 远距离抛射物品（跨区块） |

### v0.4.3+ 新增的网络智能

| 组件 | 功能 |
|---|---|
| **汽动截止阀** | 嵌入导管链，红石控制通断——终于有了管道开关 |
| **汽动管道模块** | 嵌入导管链中段的三合一模块 |
| 　└ 流量计模式 | 统计过站物品速率 → 0–15 模拟红石输出 |
| 　└ 限流阀模式 | 限制每窗口通过的物品数，超额自动断网 |
| 　└ 溢流报警模式 | 下游满载时输出红石报警 |

Pylon 没有物品管道的概念。蒸汽工坊的汽动网络让你可以把 20 台机器用导管连成一张网，物品自动从入口流向出口，不需要任何漏斗链。

## 三、蒸汽逻辑：信号就是蒸汽本身

这是蒸汽工坊最独特的设计：**蒸汽压力本身就是逻辑信号**。

不像传统工业模组的"机器→红石→电路→机器"路线，蒸汽工坊的逻辑元件直接读取相邻机器的蒸汽量作为 HIGH/LOW，逻辑判定通过后，把**真实的蒸汽**从输入面物理搬运到输出面——你的信号线传导的不是"电"，而是可以真正做功的蒸汽。

### 逻辑元件家族

| 元件 | 功能 | 类比 |
|---|---|---|
| **汽动逻辑门** | AND/OR/XOR/NOT 四模式；比较器式三输入一输出 | 红石逻辑门 |
| **汽动差压门** | A − B ≥ 阈值时放行；防止回流 | 比较器减法模式 |
| **汽动脉冲器** | 蓄能→阈值→释放定量脉冲 | 红石脉冲器 |
| **汽动锁存器** | SET/RESET，无信号维持上次状态 | RS 锁存器 |
| **蒸汽振荡器** | 持续蓄压→上阈值喷发→下阈值再蓄压 | 红石时钟 |
| **蒸汽压力传感器** | 蒸汽填充比 → 0–15 模拟红石 | 比较器 |
| **蒸汽差分机** | 可编程 3 阶差分序列发生器 | *没有对应物* |

最后一个——**蒸汽差分机**——致敬了巴贝奇差分机。它能生成等差、抛物线、三次曲线的序列信号，用来让产线节奏渐快渐慢、或者生成非线性供汽波形。这在任何其他 Minecraft 工业模组里都没见过。

所有逻辑元件都有朝向，放置即定向（绿色端点 = 输入，红色端点 = 输出），无需手动配线。门与门之间可以直接串联——因为每个门自带蒸汽缓存，输出的蒸汽可以直接被下一个门读取。

## 四、蓝图产线：真正的流水线

Pylon 的每台机器是独立个体。蒸汽工坊的**产线系统**让你把多台机器串成一条有序的自动流水线：

```
产线入口 → 蒸汽压力炉 → 蒸汽研磨机 → 精密铸造炉 → 产线出口
   ↑                                                    ↓
  原料自动进入                                      成品自动输出
```

- **产线蓝图**：一个手持物品，定义了机器的排列顺序和配方
- **产线入口**：14 格缓冲（7 原料 + 7 燃料），支持汽动网络自动补料
- **产线缓冲箱**：产线中间的暂存点
- **产线出口**：成品收集，可接汽动网络自动搬走
- **工序中间品**：带数据的半成品物品，沿产线依次传递，耐久条显示当前工序进度
- **产线注册表**：全局 YAML 索引，方便服主查阅和管理所有活跃产线

产线兼容多种机器类型：蒸汽工坊的所有加工机器、Pylon 的搅拌锅、原版熔炉/高炉/烟熏炉、原版工作台。这意味着你可以在一条产线里混用蒸汽工坊机器和原版机器。

**自动产线模块**更进一步：把升级模块插入产线入口，产线就能自动扫描下游机器并推送物品，不需要蒸汽臂。

---

## 五、蒸汽装备：穿在身上的工业

蒸汽工坊的装备系统和 Pylon 的工具完全不同——它引入了**可拆卸蒸汽罐**的概念：

- **蒸汽罐工作台**：在装备上安装/拆卸蒸汽罐
- **蒸汽充汽室**：给已安装罐体的装备充汽
- 三种罐体材质（黄铜/因瓦/钨），容量递增
- 装备没有内置能量——**没罐子就是一堆废铁**

### 套装技能

四种材质、四种专属技能：

| 套装 | 技能 | 效果 |
|---|---|---|
| 黄铜 | ⚡ 蒸汽冲刺 | 瞬移短距离 |
| 因瓦 | 🔥 熔炉吐息 | 范围火焰 |
| 锰钢 | 🛡 蒸汽护盾 | 5 秒伤害吸收 |
| 钨 | 💥 蒸汽地震 | 范围 AOE + 击退 |

### 蒸汽飞行

钨套装胸甲解锁**喷气推进**：

1. 蹲跳起飞 → 点火阶段（3 秒倒计时）→ 推进飞行
2. 飞行中可切换为**悬浮自由飞行**模式（消耗更高）
3. 落地自动切回喷气模式
4. 蒸汽耗尽 → 悬浮中断 → 自由落体

## 六、升级与调校

蒸汽工坊的所有加工机器都支持**通用升级模块系统**：

| 升级模块 | 效果 |
|---|---|
| 节能模块 | 降低每次加工的蒸汽消耗 |
| 自动输入 | 从相邻容器自动抽取原料 |
| 自动输出 | 成品自动推送到相邻容器 |
| 加速模块 | 缩短加工周期 |
| 批量模块 | 单次加工多份 |
| 范围模块 | 扩大作用范围 |
| Pylon 兼容 | 让蒸汽工坊机器处理 Pylon 原版配方 |

**机器调校仪**可以微调单台机器的参数，所有数值都可通过 YAML 全局调整。

---

## 七、Pylon 深度集成

蒸汽工坊不是一个孤岛。它深度集成了 Pylon 生态：

- **Pylon 配方兼容**：通过 Pylon 兼容升级模块，蒸汽压机可以执行 Pylon 的锤击配方，蒸汽研磨机可以执行 Pylon 的研磨石配方，蒸汽压力炉可以执行 Pylon 的窑炉配方……共 8 种 Pylon 配方类型可在蒸汽机器上运行
- **Pylon 万能涡轮**：专门为 Pylon 原生机器设计的涡轮，可以贴着 Pylon 机器加速
- **产线兼容 Pylon**：产线系统支持 Pylon 的搅拌锅作为产线成员
- **研究树互补**：蒸汽工坊的研究不覆盖 Pylon 的解锁体系，两套研究并行

---

## 八、研究与学科：不是一上来全解锁

蒸汽工坊的内容通过**研究树**渐进解锁，分四个层级：

| 层级 | 研究点 | 门控 | 内容 |
|---|---|---|---|
| T0 入门 | 15–30 | 无 | 基础金属、植物加工 |
| T1 初期 | 50–70 | 无 | 基础蒸汽机器 |
| T2 中期 | 100–160 | 无/学科 | 调校、装配、产线、逻辑、物流 |
| T3 高级 | 120–200 | 学科 | 高阶合金、蒸馏、精密加工 |
| T4 终局 | 300–600 | 无 | 涡轮、蒸汽装备、蒸汽护甲 |

**学科点数**（★ 标注的研究）需要在**蒸汽科研接口**中消耗四种学科点数（材料学/生物学/精密工程/化学）才能解锁——不只是攒够研究点就行，你还得用正确的机器生产分析样本来积累学科点数。
