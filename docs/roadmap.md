## 阶段 1 — 精密升级与 Pylon 联动

### #5 精密升级模组系统
- 手持画刷右键机器 → 打开独立升级 GUI，与主操作界面分离
- 5 种模组：自动进料 / 自动出料 / 超速 / 节能 / 批量处理
- 模组通过合成或精密加工获得

### #6 蒸汽产线系统（Production Line）

#### 概念
将多台机器与入口/出口方块串联成一条直线产线。组内机器屏蔽所有外部气动端口，物品只沿产线方向单向流动；入口/出口仍对接外部气动货运网络，作为产线的边界节点。

```
[入口] → [机器A] → [机器B] → [机器C] → [出口]
   ↑                                        ↓
外部货运网络                           外部货运网络
```

#### 布局约束
- 入口、所有机器、出口必须严格共线（仅 X 轴或 Z 轴）
- 紧密相邻，中间不允许有空格

#### 新增内容
- **蓝图道具**（`ProductionLineBlueprint`）：手持右键各方块完成产线配置
- **产线入口**（`ProductionLineInlet`）：接收外部气动物品，推送给产线第一台机器
- **产线出口**（`ProductionLineOutlet`）：接收末台机器产物，输出到外部气动网络

#### 数据存储
- 产线 ID（UUID）存入 PDC，入口/出口/各机器共享同一 ID
- 机器在线内的序号也存 PDC，决定物品流方向

#### 物品流动
- 每台机器完成配方后，产物自动推入下一台输入槽（推式）
- 若下一台输入槽已满或配方不匹配，产线在该节点**卡住**等待，上游机器继续持有产物不丢弃

### #7 Pylon 联动模组
- 特殊升级模组，插入后蒸汽机可执行 Pylon 配方
- 使蒸汽动力成为 Pylon 手动/液压机的替代驱动

### #4 高阶合金消耗配方
- 给现有高阶合金（钨、因瓦、杜拉铝、锰钢、锰青铜、钯合金）
  补充下游消耗配方，避免只能合成没有用途

## 阶段 2 — 红石可编程控制层

### #8 机器侧通用红石接口
- 所有精密机器侧面可安装红石接口模块
- 支持：运行中信号输出 / 满载信号输出 / 外部使能输入

### #9 精密红石控制器（分层可编程）
**基础层（低门槛，开箱即用）：**
- 预设模式选择：计时脉冲 / 物品阈值触发 / 比较输出
- GUI 内简单参数调节（时间间隔、触发数量）

**高级层（高门槛，插入编程核心）：**
- 专用槽位插入"精密编程核心"（高阶合金合成，稀缺）
- 解锁多通道逻辑编辑、条件嵌套、变量寄存器
- 编程核心本身是消耗品或可取出重编程

### #10 精密物品计数器 + 检测器
- 计数器：统计通过气动管道的物品数量，输出模拟红石信号
- 检测器：检测容器/管道中的特定物品，支持 PDC 识别（区分同 material 的 Rebar 物品）

### #11 精密逻辑元件
- 精密锁存器（SR / D 型）
- 精密比较器（支持模拟量）
- 精密路由器（根据条件分流气动管道）
- 精密总线（多信号复用到单通道）

## 阶段 3 — 终末多方块

### #12 精密真空炉 + 精密结晶塔（多方块）
- 精密真空炉：需要真空环境，处理高纯度冶炼配方
- 精密结晶塔：液体 → 晶体的慢速转化，需要维持稳定温度

### #13 量子稳定室 + 量子核心（终末）
- 量子稳定室：多方块结构，提供"量子稳定"环境加成
- 量子核心：终末合成目标，需要前序所有工序链产物

### #14 精密蒸汽透平（桥接到 Pylon 电力）
- 高压蒸汽 → Pylon EU 的转换机器
- 使蒸汽体系成为 Pylon 电力的合法来源之一
